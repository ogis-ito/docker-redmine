# docker-redmine
docker-compose settings for docker-redmine

## 利用方法

    $ git clone https://github.com/ogis-ito/docker-redmine.git
    $ cd docker-redmine
    $ cp .env.example .env
    $ docker-compose build
    $ docker-compose up -d
